# 📊 E-Commerce Sales Analysis Project

## 🔹 Overview

This project focuses on end-to-end data analysis of an e-commerce dataset using **SQL, Python, and Power BI**.
The goal is to extract meaningful insights related to sales performance, customer behavior, and product trends.

The project covers the complete data pipeline:

* Data Loading & Cleaning (Python)
* Data Analysis (SQL + Python)
* Data Visualization (Power BI)
* Reporting & Presentation

---

## 🔹 Dataset

* Source: Superstore Sales Dataset
* Total Records: ~10,000
* Features:

  * Order Details (Order ID, Date, Ship Date)
  * Customer Information (Customer ID, Segment, Region)
  * Product Details (Category, Sub-Category, Product Name)
  * Metrics (Sales, Quantity, Discount, Profit)

---

## 🔹 Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (MySQL / PostgreSQL / SQL Server)
* **Power BI** (Dashboard & Data Visualization)
* **Jupyter Notebook**
* **Gamma** (Presentation Creation)

---

## 🔹 Project Steps

### 1. Data Loading & Cleaning (Python)

* Loaded dataset using Pandas
* Handled missing values and duplicates
* Converted date columns to proper format
* Standardized column names

### 2. Exploratory Data Analysis (EDA)

* Sales distribution analysis
* Category-wise performance
* Monthly sales trends
* Profit vs Discount relationship

### 3. SQL Analysis

Performed business-focused queries:

* Total Revenue & Profit
* Top Categories by Profit
* Monthly Sales Trend
* Top Performing Cities
* Average Order Value (AOV)

### 4. Power BI Dashboard

Created an interactive dashboard with:

* KPI Cards (Revenue, Profit, Profit Margin, AOV)
* Sales Trend (Monthly)
* Category-wise Performance
* Segment-wise Revenue Distribution
* Filters (Category, Ship Mode, Date)
